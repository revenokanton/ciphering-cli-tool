# Ciphering CLI Tool

CLI tool for encoding and decoding a text by 3 substitution ciphers:
- Caesar cipher
- Atbash cipher
- ROT-8 as variation of ROT-13

###Installation

You will need Node `>=14.17.0 <15` and NPM `>=7.15.0` installed locally.

Open the project

```
cd ciphering-cli-tool
```

Install dependencies
```
npm install
```
